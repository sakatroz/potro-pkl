2.CSS

A.Macam macam style CSS
  • Mengedit munggunakan selector style - /style di head(embed style)
  • Mengedit didalam kurung selector(inine style)
  • Mengedit dengan link terpisah (link rel="...." href="nama file.css")di head (external style)
B.Class dan id

1.Menulis class di dalam selector berarti menulis titik(.) beserta nama
contoh(p class="satu")(/p) berarti menuliskan .satu di style head

2.Menulis id di dalam selector berarti menulis # beserta nama
contoh(p id="dua")(/p) berarti menuliskan #dua di style head

C.Div dan Span

Agar lebih mudah menggunakan class dan id
contoh (div class="satu")(p)(span)belajar(/span) div dan span(/p)
-jika ingin mengedit (p) ketik .satu p di style head
-jika ingin mengedit belajar ketik .satu span di style head
